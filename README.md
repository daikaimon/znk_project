This is Japanese overview. English version is [here][1].

## znk_projectとは?
-----------------------------------

znk_project は私ども全日本ネットワーク研究所(通称ZNK)によるフリーかつオープンソースなアプリケーション
およびライブラリを開発/提供するリポジトリです.
このリポジトリで開発しているもののご紹介やドキュメント類については以下のサイトをご覧下さい.

https://mr-moai-2016.github.io
