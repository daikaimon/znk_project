# znk_project
===================================

This is Japanese overview. English version is [here][1].

## znk_projectとは?
-----------------------------------

znk_project は私ども全日本ネットワーク研究所(通称ZNK)によるフリーかつオープンソースなアプリケーション
およびライブラリを開発/提供するリポジトリです. 
このリポジトリで開発しているもののご紹介やドキュメント類については以下のサイトをご覧下さい. 

https://mr-moai-2016.github.io

## Moai Ver2.0.2 (2018/05/02 リリース)
-----------------------------------

* Moai エンジン本体

  Moai CGI に(自立実行形式ではない)インタープリタ型のCGIスクリプトを実行させた場合、
  カレントディレクトリのセッティングが行われていなかったため、多くのケースでこれが起動できないことがわかりました.
  (例えばphpスクリプトではこの問題が発生せず、rubyやpythonスクリプトなどではこの問題が発生します.)
  今回これを修正しています.

  この問題はほとんどの通常のユーザには関係ありませんが、一部のインタープリタ型のスクリプト言語を用いて
  Moai CGIを起動ならびに開発しようと試みる方にとっては、致命的な問題となります.
  このような方で既にVer2.0(最新のVer2.0.2より前のもの)をインストールされた方には申し訳ありませんが、
  このVer2.0.2以上へバージョンアップしていただくようお願いします.

Moai Ver2.0.2 およびEasterシステムの入手やインストールについては、以下を参照して下さい. 

  https://mr-moai-2016.github.io/moai2.0/install.html

既にMoai Ver2.0をインストールされている方は、<a href=https://github.com/mr-moai-2016/znk_project/releases/tag/v2.0.2>修正パッチ</a> をそれに適用することにより、
多少手早く最新のVer2.0.2へバージョンアップできます.
このパッチはこれまでの累積的な修正も全て含みます.<br>
この修正パッチは既にインストールされた Moai Ver2.0 からであれば、基本的にはどのバージョンからも適用できます.<br>
(Ver1.1.8 以前のMoaiには適用できません).<br>

## Moai Ver2.0.1 修正パッチ(2018/04/28)
-----------------------------------

* Easter(MoaiCGIアプリケーション)
  赤福と一緒に使っている場合、極一部の環境でオートリンク機能がうまく機能しないケースがあることがわかりました.
  これを修正しています.
